# Missing And Repeating

**Difficulty:** Easy  
**Accuracy:** 24.83%  
**Submissions:** 660K+  
**Points:** 2  
**Average Time:** 30m  

Given an unsorted array `arr[]` of size `n`, containing elements from the range `1` to `n`, it is known that one number in this range is missing and another number occurs twice in the array. Your task is to find both the duplicate number and the missing number.[web:25][web:29]

### Examples

**Example 1**

Input:  
`arr[] = [2, 2]`  

Output:  
`[2, 1]`  

Explanation:  
Repeating number is `2` and the missing number is `1`.[web:25][web:29]

---

**Example 2**

Input:  
`arr[] = [1, 3, 3]`  

Output:  
`[3, 2]`  

Explanation:  
Repeating number is `3` and the missing number is `2`.[web:25][web:29]

---

**Example 3**

Input:  
`arr[] = [4, 3, 6, 2, 1, 1]`  

Output:  
`[1, 5]`  

Explanation:  
Repeating number is `1` and the missing number is `5`.[web:25][web:29]

---

### Constraints

- \(2 \leq n \leq 10^6\)[web:25]  
- \(1 \leq arr[i] \leq n\)[web:25]
